# Comment obtenir python

Si vous n'avez pas python sur votre machine, vous pouvez soit suivre le guide de votre choix pour l'installer, soit utiliser pyenv.

1) Récupérez pyenv https://github.com/pyenv/pyenv et suivez ce qui y est inscrit.

2) Installez la version de python

```
pyenv install 3.12  # installe python3.12
pyenv global 3.12   # rend la version 3.12 la version de python par défaut.
```
